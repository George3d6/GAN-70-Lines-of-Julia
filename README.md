# Generative Adversarial Networks

An 70 lines [Knet](https://github.com/denizyuret/Knet.jl "Knet Github Repo") implementation of [MLP GAN](https://arxiv.org/abs/1406.2661 "arXiv") for MNIST data.

## Usage

```Shell
julia gan.jl
```
